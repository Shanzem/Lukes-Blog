TryHackMe - Ice, To aquire ROOT on a windows 7 machine which is using a out of date version of Icecast with a known CVE to exploit. To execute arbitrary code.

Recon

First thing to do is aquire recon on the target machine for this I am going to peform an NMAP scan consisting off the following options:

-sV Services running on the machine if any.
-A  Aggressive scan.
-p- Scan and list all ports.
-sS Peform a silent SYN scan.
-T4 The amount of time between each packet sent.

So I run NMAP with the following command:

sudo nmap -sV -A -p- -sS 10.10.48.89

I am now going to dig deeper into report to determin the actual build version of Icecast to see if there is an earlier and better CVE I can exploit, to which there is.

Looking at the CVE report I can determin that Icecast is running version 2.0.1 or lower which means I could also exploit CVE-2004-1561.

Description

Buffer overflow in Icecast 2.0.1 and earlier allows remote attackers to execute
arbitrary code via an HTTP request with a large number of headers.

This Expolit will allow me to gain ROOT, I will now check Metasploit to see if it has a working script.
